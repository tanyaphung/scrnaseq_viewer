# scrnaseq_viewer
This repository provides an overview of single cell RNAseq data that I have processed as part of Phung et al. 2025
Preprint: https://www.biorxiv.org/content/10.64898/2025.12.05.692533v1

# Setting up conda environment
- yaml file is provided in `resources/scrna_env.yml`
```
conda env create --name scrna --file=resources/scrna_env.yml
```
